# Auto Sales Customer Segmentation using R

This project applies data science techniques to automotive sales data to understand customer behaviour, product preferences, and order patterns. The analysis was completed as part of a Data Science using R project and focuses on three main techniques: Conjoint Analysis, K-Means Clustering, and Discriminant Analysis.

## Project Overview

The aim of this project is to convert raw automotive sales transaction data into useful business insights. The project explores how product attributes influence customer preferences, how customers/orders can be grouped based on sales behaviour, and how order status can be predicted using numerical variables.

The analysis is divided into three parts:

1. Conjoint Analysis
2. K-Means Cluster Analysis
3. Discriminant Analysis
